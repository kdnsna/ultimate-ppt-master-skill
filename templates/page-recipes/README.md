# Page Recipes

Page recipes are 4.0 structural recipes for formal decks. They sit below visual directions and above concrete PPTX/HTML builders.

Each recipe defines:

- the page job it supports
- the editable layout structure
- the visual layer policy
- the raster policy
- the main anti-patterns it prevents

Formal PPTX body pages should use editable recipe builders. Generated images may support the page as background, texture, device mockup, or icon-like visual layer, but should not replace editable text, numbers, charts, or tables.
