# Visual Directions

Visual directions are production guardrails, not decorative templates. They tell the Strategist what the deck should feel like before any page is drawn.

Use them when a deck has a recognizable delivery context. Each direction locks:

- expected audience and communication mode
- preferred page roles and rhythm
- asset expectations
- anti-patterns that make the deck look generic or unfinished

The active direction must be recorded in `design_spec.md` and `spec_lock.md`. If no direction fits, the Strategist must record `custom` and state the benchmark in one sentence.
