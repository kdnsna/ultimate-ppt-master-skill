# v6.3.5 候选说明：精修可靠性与可访问运行时

> **未发布候选。** 本文定义 v6.3.5 候选边界。它可以出现在 `main` 或 Pages 上，但不代表已有独立 tag、GitHub Release 或 marketplace 发布。

[English candidate notes](../../release/release-notes-v6.3.5.md)

## 白话更新栏

- 单页精修继续以稳定 `slideId` 为单位；未知页面或 variant 会在写入修订前被拒绝。
- 产物轮询保持单飞、可取消、页面隐藏时暂停、从当前项目恢复，并在已交付后停止。
- radio 组支持方向键、Home/End 和 roving tabindex。
- 诊断对话框捕获焦点、Escape 关闭并将焦点退回触发按钮；减弱动画用户不执行平滑滚动和非必要动效。
- 可恢复的 Bridge、URL 和修订错误在对应控件附近呈现；剪贴板失败时，手动 Agent 命令仍可见、可选中。

## 候选边界

本切片只负责单页精修验证/历史、轮询生命周期、键盘/对话框行为、减弱动画、行内错误和可见手动 Agent 兜底。Classic 仍冻结在 `?classic=1`；Desktop 只保持合同兼容。

## 发布前证据

- 未知 `slideId`/variant 拒绝与修订历史测试；
- 纯键盘 radio/对话框验收与 reduced-motion 检查；
- 刷新、隐藏页、过期请求与已交付轮询测试；
- 离线、无 Agent、可启动、生成中、成品完成与质量阻断六种状态均只有一个明确主操作。

## 独立回滚边界

回滚精修/轮询/可访问性分组，不回滚 v6.3.4 项目/会话身份和 v6.3.3 证据真实性。已持久化 `slideId` 与修订文件必须继续可读，Classic 兼容入口在回滚全程中必须保持可用。

版本化发布前，应把本分组表示为独立提交或明确的 revert map。候选标记本身既不创建，也不证明已有独立 v6.3.5 tag。
