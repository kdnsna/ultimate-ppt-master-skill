## 推荐路线

```text
打开 Web Experience -> 拖入资料或粘贴摘要 -> 启动 npm run bridge -> 发送到本地 Bridge -> 把 handoff 项目交给 Codex / Claude Code / Hermes / OpenClaw 深加工
```

v4.0 的新增重点是：先用 page recipes 锁定页面结构，再用无文字 visual layers 增强观感，最后用审计阻止重复模板和正式正文页整页图片化。
