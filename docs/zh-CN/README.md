## 推荐路线

```text
打开 Web Experience -> 拖入资料或粘贴摘要 -> 写一句目标 -> 启动 npm run bridge -> 创建本地项目 -> 执行复制好的 Codex 命令
```

v4.4 的新增重点是：网页入口从复杂控制台改成 Codex 优先启动器。v4.3 的渲染审阅、v4.2 的 DeckIR 页面地图和 v4.0 的可编辑视觉契约继续作为底层主线。
