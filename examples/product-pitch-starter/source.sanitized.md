# 产品路演公开案例 · 脱敏合成资料

## 主题

Ultimate PPT Master 产品路演。

## 受众

合作伙伴、早期用户、产品团队与潜在支持者。

## 核心信息

有用的 AI 演示工作流必须能够处理真实资料、保留可编辑输出，并让本地 Agent 生产出可以检查的成品。

## 产品切入点

- Web Experience 帮助用户说清目标、加入资料并确认交付路线。
- Bridge 在本地创建带 manifest、故事板和恢复状态的项目。
- Agent Skill 完成 PPTX 或 Web Deck 的正式生产与质量复核。

## 证明点

- 支持 Markdown、PDF、DOCX、PPTX、XLSX、URL 与粘贴文字。
- 保留 source、manifest、storyboard、source map 与质量报告。
- 稳定预设减少从空白提示开始的不确定性。

## 预期输出

- 主交付：用于产品叙事的杂志风 Web Deck。
- 可选交付：供合作伙伴评审的可编辑 PPTX 附录。
