01 / HookPrompt box → execution loop

What changed

热点不是一个模型名，而是一整套工作方式变了。

Before
Ask

用户把需求塞进一个输入框，等模型一次性回答。

Now
Act

Agent 拆任务、读文件、跑命令、看结果，再继续修复。

Next
Ship

真正的产品价值在可复现交付，而不是一次性生成文本。
