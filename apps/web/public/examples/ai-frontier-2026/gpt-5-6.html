06 · Availability BoundariesTrust requires limits

正式发布不等于：所有入口、模式和任务都完全相同

现在能确认

  • GPT-5.6 已在 ChatGPT、Codex 与 OpenAI API 正式提供。
  • Sol / Terra / Luna 是可独立演进的官方能力层级。
  • 官方公布三档价格，并提供 max 与 multi-agent / ultra 路线。
  • API 支持 Programmatic Tool Calling 与 multi-agent beta。

现在不能假设

  • 不同订阅层级并不一定拥有相同模型与 effort 入口。
  • 官方 benchmark 不能直接代替自己的真实任务评测。
  • 多代理与更高 effort 会增加 token 消耗和治理成本。
  • 生产选型仍需验证正确率、时延、返工与安全边界。
