# Product Pitch Starter Source

## Topic

Ultimate PPT Master product pitch.

## Audience

Investors, partners, early customers, and product teams.

## Core Message

A useful AI presentation tool must handle real source files, preserve editable outputs, and let a local Agent produce something checkable.

## Product Wedge

- Web Experience helps users describe the goal and prepare source material.
- Bridge writes the handoff folder locally.
- Agent Skill performs final PPTX or Web Deck production and QA.

## Proof Points

- Supports Markdown, PDF, DOCX, PPTX, XLSX, URL, and pasted notes.
- Keeps `source.md`, `manifest.json`, `engine-plan.md`, and `quality-checklist.md` inspectable.
- Preset starter packs reduce blank-prompt uncertainty.

## Required Output

- Primary: magazine-style Web Deck for product storytelling.
- Optional: editable PPTX appendix for partner review.
