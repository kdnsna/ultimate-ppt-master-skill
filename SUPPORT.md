# Support

Use the route that matches your need:

- Setup help: open a `Setup or configuration help` issue.
- Output quality: open a `Generation quality` issue with sanitized examples.
- Bug reports: open a `Bug report` issue with reproduction steps.
- Ideas and workflow questions: use GitHub Discussions.

Before posting, remove private document text, API keys, exact budgets, names, internal paths, and production logs.
